## 參考程式使用方法

1. 將橡筋裝填在砲台的齒輪上。
2. 將電源模型打開。
3. 利用手掣上的搖桿控制機械車的上下左右移動。
4. 按Y切換至轉向模式，控制機械車的角度。
5. 按A和B調較炮台的角度。
6. 按X發射橡筋。
