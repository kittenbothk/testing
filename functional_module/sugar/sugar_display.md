# Sugar二期 數碼顯示模組

![](./images/display_render.png)

這是一個數碼顯示模組，可以用來顯示數字或時間等啲信息。

## 產品參數

接口: I2C

## 產品接線

用4PIN連接線將模組連接到Robotbit上的I2C接口。
