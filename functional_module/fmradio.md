# FM收音機模組

這是一隻收音機模組，它可以接收FM電台訊號。

它使用標準的3.5mm插頭，可以使用3.5mm的耳機或揚聲器作輸出。

## 產品參數

- 工作電壓：2.5V~5V
- 類型：i2c模組
- 接口：4pin杜邦線
- 有效頻率：76MHz~108MHz
- 天線接口：3.5mm
- 輸出接口：3.5mm

## 接線教學

將FM收音機模組連接到Robotbit的i2c接口。

    SCL接SCL，SDA接SDA。
