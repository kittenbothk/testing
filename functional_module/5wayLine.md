# 五路巡線模組

五路巡線模組(HKBM8003A)

![](./images/5wayline2.png)

這是一隻巡線模組，它有五顆巡線感應器，它返回的數值是類比形式。感應器收到的光度越少（遇上黑線），返回的數值就會越低。

模組採用五通道ITR20001/T紅外線探測器。靈敏度高，抗干擾。性能穩定。

## 產品參數

- 工作電壓：5V
- 感應器間距：11mm
- 固定孔直徑：5mm
- 感應距離：1cm-5cm
- 接口：杜邦線

## 接線教學

五路巡線模組可以接3v或5v電源。

    由於這是類比模組，所以只能使用robotbit的PIN0-2。（使用PIN0的話需要拔除蜂鳴器跳線帽）
