# 雙路IR巡線魔塊

雙路IR巡線魔塊 (HKBM8012B)

![](./images/07_03.png)

這是一個紅外線巡線魔塊，它有組兩路尋線感應器，偵察到黑線後魔塊背面的LED會熄滅，觸發尋線事件。

![](./images/IMG_2570.GIF)

## 詳細介紹

![](./images/07_01.png)

## 產品參數

- 支援電壓：3V-5V
- 尺寸：56mm X 24mm X 25mm
- 接口：4pin防反插接口

## 注意事項

- 巡線魔塊需要盡量貼近地面。
- 枱面或地板必須為不反光的表面。

## 接線方法

將巡線魔塊用4pin排線連接至Armourbit。

![](./kbimages/07_02.png)

### 黑線檢測

我們在枱面貼上黑色膠紙，用巡線魔塊檢測，當感應器A或B檢測到膠紙的時候就會分別顯示笑臉或哭臉，假如沒有偵測到線條就會顯示心形。

![](./images/linefollow.png)
