# 雨滴水位感應器

雨滴水位感應器 (HKBM8012G)

![](./images/03drop.png)

這是一塊雨滴水位感應器它可以測量水位高低和是否有雨滴，水位越高數值越大。

![](./images/IMG_2581.GIF)

![](./images/IMG_2578.GIF)

## 使用注意事項：

- 若長期不使用感應器，請妥善放置乾燥地方保管，防止潮濕被氧化。
- 感應器上有最大限位線，請勿過度插入土壤或者水中，避免損壞電路。

## 接線方法

將溫度濕度感應器用4pin排線連接至Armourbit，然後用3pin排線將感應器連接至溫度濕度感應器。
