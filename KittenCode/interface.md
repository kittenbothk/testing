# KittenCode操作指引

## 主介面

![](./images/interface1.png)

1. 功能區：檔案命名與儲存，模式切換，選項等
2. 文件區：專案內的文件，硬件文件系統
3. 編程區：編寫程式
4. 執行區：運行程式，上傳程式(硬件模式)
5. 控制台：顯示程式的列印，負責執行REPL模式(硬件模式)
