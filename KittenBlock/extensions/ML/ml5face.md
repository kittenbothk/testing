# Machine Learning 5 FaceAPI人臉追蹤

## 加載Machine Learning 5插件

由於今次教程需要使用視像偵測，所以我們要加載視像偵測插件。

按下左下角的插件加載按鈕。

![](../images/add.png)

選擇Machine Learning 5和視像偵測插件。

![](../images/add2.png)

最後再添加畫筆插件。

![](../images/ml28.png)

加載成功！

![](../images/ml6.png)

電腦配置比較弱(例如：缺乏顯示卡)的用戶，可以點選CPU模式提升效能。
